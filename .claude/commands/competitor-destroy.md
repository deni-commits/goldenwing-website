# /competitor-destroy - Konkurrenz zerstören

## Zweck
Analysiert einen Konkurrenten komplett und erstellt einen Angriffsplan um dessen Rankings zu übernehmen.

## Workflow

### 1. Domain-Analyse des Konkurrenten
```
semrush_domain_overview: Domain-Metriken
semrush_domain_organic_keywords: Top Rankings
semrush_traffic_summary: Traffic-Daten
semrush_traffic_sources: Woher kommt Traffic
```

### 2. Keyword-Gap finden
```
semrush_competitors: Gemeinsame Konkurrenten
Vergleiche: Konkurrent Keywords vs. GoldenWing Keywords
Finde: Keywords wo Konkurrent rankt, wir nicht
```

### 3. Backlink-Analyse
```
semrush_backlinks: Backlink-Profil
semrush_backlinks_domains: Verweisende Domains
Finde: Hochwertige Backlinks die wir auch bekommen können
```

### 4. Content-Gap
```
Analysiere Top-Seiten des Konkurrenten
Finde Themen die wir nicht abdecken
```

## Output: Angriffsplan

Erstelle einen konkreten Plan mit:

1. **Quick Wins** - Keywords mit niedriger Difficulty wo wir schnell ranken können
2. **Content zu erstellen** - Neue Seiten die wir brauchen
3. **Backlinks zu holen** - Liste von Domains für Outreach
4. **Bestehende Seiten optimieren** - Welche GoldenWing-Seiten verbessern

## Verwendung
```
/competitor-destroy webdesign-agentur.at
/competitor-destroy [konkurrent-domain]
```

## API Units
Ca. 200-400 Units pro Analyse (Domain Overview, Keywords, Backlinks, Traffic)
